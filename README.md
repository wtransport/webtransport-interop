## Overview

An automated infrastructure for evaluating WebTransport semantics, conformance, and cross-implementation interoperability.

The testing matrix validates native protocol behaviors, focusing on:

- State machine boundaries and strict flow control.
- Bidirectional and unidirectional stream multiplexing.
- Unreliable datagram transmission and path resilience.
